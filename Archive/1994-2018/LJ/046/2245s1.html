Resources

PostgreSQL: http://www.postgresql.org/ and ftp://ftp.postgresql.org/pub/

Java JDBC drivers:
http://www.hottub.org/~adrian/Java
http://www.demon.co.uk/finder/postgres/index.html
ftp://ftp.ai.mit.edu/people/rst/rst-jdbc.tar.gz

PostODBC: http://stud1.tuwien.ac.at/~e9025461/
(free ODBC-Driver for PostgreSQL. Current: v0.21)

OpenLink ODBC: http://www.openlinksw.com/
(commercial ODBC-Driver with free test version up to two simultaneous connections)

mpsql: http://www.troubador.com/~keidav/mpsql.html
(graphical [Motif] query interface)

mpmgr: http://www.troubador.com/~keidav/mpmgr.html
(Motif database management tool for PostgreSQL, preview version)

Kanchenjunga: http://www.man.ac.uk/~whaley/kj/kanch.htm
(Alpha version of a Java rapid application development tool for PostgreSQL)

Appgen: http://www.man.ac.uk/~whaley/ag/appgen.html
(Alpha version of 4GL for web databases)

ECPG: ftp://ftp.lysator.liu.se/pub/linux
(Alpha version of an embedded SQL C-preprocessor)

Dbf2SQL: ftp://sti1.uni-duisburg.de/pub/databases/dbf2sql/
(converter to/from dBase)

haspureX: http://www.csr.unibo.it/~abaldoni/haruspex.html
(Gimp extension for a picture database in PostgreSQL)

Announcements: subscriptions to pgsql-announce-request@postgresql.org

User Questions: subscriptions to psql-questions-request@postgresql.org

Bugs: subscriptions to psql-bugs-request@postgresql.org

Ports: subscriptions to psql-ports-request@postgresql.org

Other mailing lists, in particular for developers, can be found at the PostgreSQL web site.

Both listings that appear with this article are available by anonymous download in the file ftp://ftp.ssc.com/pub/lj/listing/issue46/2245.tgz.
