Resources

The BadRAM patch is not integrated in the kernel (yet), but it can be found at
http://home.zonnet.nl/vanrein/badram/.

A more extensive example of using the BadRAM patch can be found in the patched kernel, in Documentation/badram.txt/.

The benchmarks on a patched system are on
http://home.zonnet.nl/vanrein/badram/benchmarks.html.

The Memtest86 program can be found on
http://reality.sgi.com/cbrady_denver/memtest86.

The slab allocator is described in Jeff Bonwick, “The Slab Allocator”: An object-oriented kernel memory allocator, in the summer 1994 USENIX proceeding.
