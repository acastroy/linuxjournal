It wouldn't be difficult to write a program which translates add 3 to x. to X += 3;.

But how would you translate a start statement? Or the sort verb? You would translate it into a function call. Then you need a function library capable of handling ISAM files with multiple and split keys. Record locking would be nice, also. Sequiter Software and Byte Designs will happily sell you a library like this. If you would like to develop the library yourself, rather than pay good money for it, you will quickly discover that the code translator is only the tip of the iceberg.
