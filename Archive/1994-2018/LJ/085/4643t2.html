Table 2. Header Field Classifications 
Static Known      Never needs to be communicated.

Static--Def       Stream defining field treated as static.

Static Constant   Must be communicated at least once.

Changing          Seemingly random and unpredictable, must be sent as is.

Inferred          Values that can be obtained from other values in the header.
