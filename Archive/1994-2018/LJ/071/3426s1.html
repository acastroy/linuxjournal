The NOCOL Suite

This is a list of the more commonly used monitors which form NOCOL.

Monitor: What it monitors

apcmon: APC Smart UPSs armon: Appletalk routing tables
bgpmon: BGP processes on Cisco routers
bpmon: bootp service monitor
ciscomon: Cisco routers (CPU usage, airflow etc.)
hostmon: Unix host conditions (CPU usage, disk space, load etc.)
ippingmon: Whether a host is responding to TCP/IP pings
modemmon: Modem line usage on Cisco terminal servers
novellmon: Novell service monitor
nrmon: Novell routing tables
nsmon: Name servers (DNS)
ntpmon: NTP servers
rpcpingmon: RPC portmapper on hosts
snmpmon: SNMP data on hosts
syslogmon: Log files for specified expressions
tpmon: Network throughput
