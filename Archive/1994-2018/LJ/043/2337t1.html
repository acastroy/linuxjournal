Table 1. Support Packages Needed by MagicFilter

Package Type of Printing Support Provided
gcc compiles MagicFilter.
magicfilter Prints text files; fixes “staircase effect”; squelches unprintable files.
ghostscript Prints PostScript files.
netpbm Handles most graphics files.
nenscript/enscript Prints text to a PostScript-only printer.
dvips Converts TeX .dvi files to PostScript.
groff Converts troff files (e.g., man pages) to PostScript.
sendmail/other mailer Receives error messages from MagicFilter and sends them to user.
