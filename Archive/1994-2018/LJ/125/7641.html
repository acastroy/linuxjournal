Resources for “WordPress”

Reuven M. Lerner

Issue #125, September 2004

WordPress is available from the project's home page, www.wordpress.org. However, you can download the latest code in .tar.gz format from wordpress.org/latest. I found this to be an especially clever and useful way to ensure that a single URL always has the latest version. The WordPress site includes documentation, active support forums and a Weblog describing the latest development efforts.
