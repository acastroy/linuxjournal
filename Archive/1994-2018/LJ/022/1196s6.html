Msort is an slocal replacement written in Perl. Configuration uses the same .maildelivery file that MH's slocal does, but allows for Perl-strength regular expression matching. It is written by Eric Berggren at Portland State University, Portland, Oregon.

Send e-mail to jay@fork.com for a freely avail able copy of Msort.
