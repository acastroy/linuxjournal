Listing 2. Simple isolinux.cfg File

default Distro
prompt 1
timeout 100
F1 f1.txt
F2 f2.txt
label Distro
  kernel kernel/vmlinuz
  append initrd=initrd.gz load_ramdisk=1 ramdisk_size=5000 rw root=/dev/ram0
label hd
localboot 0x80

f1.txt sample file

Welcome to my Distro!
Available boot labels:
Distro (default): boot from CD
hd: boot from HD
