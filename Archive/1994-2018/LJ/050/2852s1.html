Credits and Resources

I've taken information from a couple of sources for this article. I'd like to thank Ambrose Au, the author of the IP Masquerading Mini-HOWTO for the examples of configuration lines for startup scripts to initialize networking correctly. Also, I'd like to thank Greg Alexander for enlightening me on the set bit netmask notation, and, the authors of the IP Masquerading option in the kernel, Pauline Middelink, Juan Jose Ciarlante, Richard Lynch, Nigel Metheringham and Keith Owens. Thanks for writing this wonderful feature of the Linux kernel. Also, the special IP Masquerading modules for certain protocols deserve some credit; check out the source code for the names of all the hard working individuals. Other sources of information about IP Masquerading can be found at the following URLs:

http://sunsite.unc.edu/linux/HOWTO/HOWTO-INDEX-3.html

http://www.wiznet.ca/~ambrose/
