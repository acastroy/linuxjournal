Credits

The MD driver, the linear and RAID-0 personalities code were written by Marc Zyngier (maz@gloups.fdn.fr) and is part of the Linux 2.0 kernel distribution.

The MD extensions described in this report, RAID-1, RAID-4 and RAID-5 personalities code were written by Gadi Oxman (gadio@netvision.net.il), Ingo Molnar (mingo@pc5829.hil.siemens.at) and Miguel de Icaza (Miguel.de.Icaza@Linux.Org).

The new personalities of the MD driver were originally tested using the loop-back driver to simulate the disk array with files.

This article was first presented as a speech by Miguel de Icaza at Linux Expo, April 5, 1997.
