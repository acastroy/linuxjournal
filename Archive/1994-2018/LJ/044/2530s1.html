Dialog Windows

New/Open: Create a new image window of specified dimensions, image type (color or grayscale) and initial fill type (white, background or transparent), or open an existing image in one of the supported file formats.

Preferences: Set the number of levels of undo and the type of background used to signify transparent regions of an image.

Tool Options: Each tool available from the Toolbox has a Tool Options dialog that allows the user to select configurable parameters for the given tool. This window is opened by double clicking the tool icon in the Toolbox. Not all tools have configurable options, so for some tools this dialog simply states that there are no configurable options.

Brushes: Allows the selection of a brush type for use with the paintbrush, pencil and eraser tools.

Patterns: Allows selection of a pattern to be used with the Bucket tool when the bucket-fill type is set to Pattern Fill.

Color Palette: Allows selection of colors for various tools and processing filters. The Color Selection dialog, shown in Figure 2, permits the selection of colors through the use of a color wheel or slider values.

Layers and Channels: Allows selection and manipulation (move, delete, add, scale, merge, etc.) of layers and provides access to the individual channels for the current image.

Gradient Editor: Permits the creation, modification, use and deletion of gradients. Gradients are used with the Gradient tool that is selectable from the Toolbox.
