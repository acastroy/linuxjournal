TIS Resources

The TIS Firewall Toolkit can be obtained from ftp://ftp.tis.com/pub/firewalls/toolkit/fwtk-v1.3.tar.Z

The TIS Firewall Toolkit documentation is found at ftp://ftp.tis.com/pub/firewalls/toolkit/fwtk-doc-only.tar.Z

The TIS Firewall Toolkit Users Archive is at ftp://ftp.tis.com/pub/firewalls/toolkit/fwall-users-archive/

The Linux patches for the fwtk can be found at ftp://ftp.tisl.ukans.edu/pub/security/firewalls/fwtkpatches.tgz

S/Key version 2.2 for Linux can be found at http://sunsite.unc.edu/pub/Linux/system/Misc/skey-2.2.tar.gz

Tripwire is a file integrity checker and can be found at ftp://coast.cs.purdue.edu/pub/COAST/Tripwire/tripwire-1.2.tar.Z
