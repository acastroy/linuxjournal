Resources

CPU Affinity Patches for Both the Kernel and glibc: www.kernel.org/pub/linux/kernel/people/rml/cpu-affinity.

Schedutils, a suite of scheduler-related tools: tech9.net/rml/schedutils. An included tool, taskset, provides a powerful interface for manipulating CPU affinity.

The latest procps contains a version of ps that can display the processor on which a process is running: sources.redhat.com/procps.

Red Hat—their latest release, 9, ships with full support for the CPU affinity system calls: www.redhat.com.
