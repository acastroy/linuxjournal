Listing 1. Chklogs Configuration File

#  Chklogs configuration file for host panama
#  (c)1995,1996,1997 D. Emilio Grimaldo T.
#:options global
#:global  /var/log/Chklogs
/var/log/messages 7d archive  4
/var/log/xferlog  1m execute ck_xferlog %L
/var/log/xferlog  1m truncate
#:Group NetNews
#:Pre  /usr/lib/news/ctl_news stop
#:Post  /usr/lib/news/ctl_news continue
/var/log/news  100K  archive  3
/var/log/news-errors 25000archive 1
#ENDGROUP
/var/log/httpd/log execute /your/own/scanner -opt\
   A -opt B
/var/adm/uucp/ execute ck_uutraf %L
