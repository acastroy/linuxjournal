Specs

SlotServer 1000

  • x86-compatible, 300MHz
  • Up to 512MB of memory
  • Gigabit Ethernet to the PCI bus
  • External RJ45 10/100 Ethernet interface
  • Internal Gigabit Ethernet
  • Two MAC addresses

Additional Features for the SlotServer 3000

  • Intel Pentium III x86 processor, 700MHz
  • Up to 2GB of memory
  • ECC and non-ECC memory support
  • Up to four MAC addresses
  • Up to three external 10/100 Ethernet ports
  • Optional VGA, USB, audio interface
