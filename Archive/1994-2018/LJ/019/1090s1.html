Text-only Resources

The home sites for some text-only resources are listed below. When a double site is specified, the first one is the true one, useful when you look for alpha version, and the second is the widely mirrored one. When a single site is specified, it will include alpha releases and is a widely mirrored one.

  • gpm iride.unipv.it:/pub/gpm sunsite:/pub/linux/system/Daemons
  • kbd ftp.funet.fi:/pub/OS/Linux/PEOPLE/Linus sunsite:/pub/Linux/system/utils/console
  • SVGATextMode sunsite:/pub/Linux/system/utils/console
  • Emacs prep.ai.mit.edu:/pub/gnu
  • jed space.mit.edu:/pub/davis/jed sunsite:/pub/Linux/apps/editors
  • dialog iride.unipv.it:/pub/gpm sunsite:/pub/Linux/apps/console
  • screen prep.ai.mit.edu:/pub/gnu
  • minicom sunsite:/pub/Linux/apps/comm
  • gnuplot prep.ai.mit.edu:/pub/gnu

As usual, the source code is the best source of information, and the kernel is the place if you are going to code some low-level software for the console. The kbd package is another good place to learn things. Remember, however, that any console-bound program is not portable, and a portable solution is better, if available.
