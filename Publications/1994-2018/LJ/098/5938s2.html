Product Information

Manufactuer: Jetico, Inc.
URL: www.jetico.com
Price:

  • Free for noncommercial use (Linux version)
  • $49.95 US for BestCrypt, Linux single-user, commercial/government license
  • $29.95 US for BCWipe, disk-erasure utility
  • $89.95 US for BestCrypt, Windows 9x/ME/NT/2K/XP single-user, commercial/government license
  • $149.95 US for BestCrypt Corporate Edition, Windows 9x/ME/NT/2K/XP single-user, commercial/government license

Requirements

  • Linux kernel 2.2.0 or later (including 2.4.x)
  • Kernel source code (installed in /usr/local/linux)
  • i386 architecture—may work on other platforms but is untested/unsupported

The Good

  • Supports Blowfish, Twofish, Rijndael, IDEA, GOST, CAST, RC6, GOST, 3DES, DES and potentially additional algorithms via kernel modules.
  • Open, modular architecture—anyone may write new algorithm modules.
  • Source code for Linux version is open, i.e., for evaluation/academic purposes.
  • Quick, simple build and installation procedure.
  • Command-line syntax similar to mount's.
  • Encrypted “containers” may be mounted by both Linux and Windows versions of BestCrypt.
  • Excellent Windows GUI; Linux GUI similar.
  • Commercial license is reasonably priced.
  • Windows Corporate Edition includes management software designed for central management of multiple desktop installations.
  • Good technical support (for commercial licenses).

The Bad

  • Requires partially built kernel sources—may be daunting for some users.
  • Linux GUI doesn't support all features of command-line tool (or of Windows GUI).
