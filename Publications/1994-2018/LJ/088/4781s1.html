The Good

  • Open edition available for free.
  • Open-source version of CLX.
  • Best programming editor ever! (Okay, I'm not an Emacs fan.)
  • Excellent web presence.
  • Unique database strategy.

The Bad

  • Installation can be troublesome, particularly for non-Linux administrators.
  • Some on-line documentation lauds Borland's products over the GNU development tools, unnecessarily.
  • No OpenGL or 3-D component.
  • Printed documentation with the product is insufficient, though available books, on-line presence and newsgroups more than make up for it.
