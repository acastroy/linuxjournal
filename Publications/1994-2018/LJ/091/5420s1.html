The UL Web Server

Eric's article covers the ULB from a desktop perspective. So when Monarch Computer Systems (one of AMD's launch partners for the Tyan Thunder K7) called us not long after they began shipping the board in their boxes, and offered to build a box to our own specifications for review, and then offered that same configuration for sale, we decided to build our own web-server configured ULB. We ordered a system with the needs of a webmaster in mind: excellent audio and video for developing and testing web applications, and sufficient capacity as a server to demonstrate and test them or fill in for the main web server if necessary. Read about the specs, suggestions and Technical Editor Don Marti's experience with our own ULB in the sidebars throughout Eric's article.
