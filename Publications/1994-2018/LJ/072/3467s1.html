  4. “Towards a High-Level Machine Configuration System”, Paul Anderson, LISA 8 Conference Paper, 1994, http://www.dcs.ed.ac.uk/~paul/Publications/LISA8_Paper.pdf/.
  5. “A Site Configuration Engine”, Mark Burgess, Computing Systems, 1995, http://www.iu.hioslo.no/~mark/research/cfarticle/cfarticle.html/.
