Process migration is where a running process is transferred from one computer to another, and keeps running, generally without the user noticing.

Fail-over is where a backup server can seamlessly take over when a main server fails; it is sometimes called “high availability”.

SMB is the TCP/IP-based protocol used by Lan Manager, Windows 95, Windows for Workgroups, and several other networking products, mostly DOS-related.
