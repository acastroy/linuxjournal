Perl and Tk

Perl is available via anonymous ftp from prep.ai.mit.edu:/pub/gnu/perl-5.xxx.tar.gz and ftp.cs.umn.edu:/pub/gnu/perl-5.xxx.tar.gz.

Tcl 7.4 & Tk 4.0 are available via anonymous ftp from ftp.smli.com:/pub/tcl/. Note that the Tcl FAQ has a list of mirror sites. You can read the FAQ on-line at http://www.cis.ohio-state.edu/hypertext/faq/usenet/tcl-faq/top.html.

CBB

CBB is available via anonymous ftp from ftp.me.umn.edu:/pub/finance/cbb-0.??.tar.gz. Because CBB is updated rather often, you are probably better off getting the latest version from the FTP site rather than using a version you find on a CD-ROM collection.
