Python 2.0

The source code in this article was written with Python 2.0, which has been available since October 2000. The latest version of Python offers many improvements, including full-garbage collection, augmented assignment operators, string methods, list comprehension, new libraries and bug fixes. I encourage anyone using Python to upgrade as soon as they can. You can find out more and download a copy at http://www.python.org/2.0/.
