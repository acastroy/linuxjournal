Mailman, the GNU Mailing List Manager

Mailing Lists for Mailman

Finally, there are a number of mailing lists you can join if you're interested in using or hacking on Mailman. Those simply interested in using Mailman at their site should join mailman-users@python.org. Developers should join mailman-developers@python.org. If you'd just like to get announcements of new releases, you should join mailman-announce@python.org, and if you'd like to get CVS check-in messages, you can join mailman-checkins@python.org. If you're interested in internationalization issues (including becoming a language translator), join the mailman-i18n@python.org mailing list. All of these mailing lists are, quite naturally, Mailman mailing lists, so visit these URLs to subscribe:

http://www.python.org/mailman/listinfo/mailman-users/

http://www.python.org/mailman/listinfo/mailman-developers/

http://www.python.org/mailman/listinfo/mailman-announce/

http://www.python.org/mailman/listinfo/mailman-checkins/

http://www.python.org/mailman/listinfo/mailman-i18n/
