Required Hardware

Linux computer with at least a 486/33, 16 MB RAM SCSI controller for CD writer
Hard disk with a free 800 MB (or more) partition CD writer (a more complete list is in the X-CD-Roast README):

  • Phillips/Kodak/IMS (CDD 521, CDD 522, CDD 2000, CDD 2600)
  • HP (4020i, C4323/C4325, 6020i)
  • Yamaha (CDR/CDE 100, CDR/CDE 102)
  • Plasmon 4220
  • Sony (CDU-920S, CDU-940S)

Optional hardware

SCSI CD-ROM
16-Bit Soundcard

Required Software

Kernel 2.0 or greater
X Window System
GNU C Compiler

Note: Earlier versions of X-CD-Roast required an installed Tcl/Tk package with the Tix extension. Starting with version 0.95b, a static linked binary of these is included.
