Resources

The mod_cntr.c module, originally written by Brian Kolaci (bk@galaxy.net) and modified by Dan Kogai (dankogai@dan.co.jp), is available at http://www.dan.co.jp/webcounter/webcounter.html.

The mSQL database, created by David “Bambi” Hughes and Hughes Technologies, is at http://www.hughes.com.au/.

To subscribe to the mSQL programming mailing list, send a message to listproc@bunyip.com with one line in the message body: subscribe msql-list Your_Name.

More information about Apache can be found on the Usenet newsgroup comp.infosystems.www.servers.unix.

Listing 1 is available by anonymous download in the file ftp://ftp.ssc.com/pub/lj/listings/issue46/2308.tgz.
