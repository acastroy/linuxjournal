The Emacs editor automatically recognizes when you are editing a file which is being kept track of (“under”) version control, and provides an interface to the version control commands so that you have to learn only a few Emacs commands (most starting with “C-x v”) in order to do version control. The Emacs manual contains documentation in the version control node. Many developers swear by this capability, and if Emacs is your favorite editor, you may quickly become addicted to it yourself.
