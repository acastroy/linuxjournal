The Test Platform

The test platform is a machine guaranteed to work with Linux, sold by one of LJ's advertisers. It has an ASUS motherboard, an AMD 486 DX/4 100 CPU, 16MB of RAM, an ET4000/W32-based video card with 1MB of DRAM, a 15 inch Chuntex monitor, a 1GB SCSI hard drive, a 4X SCSI cdrom, a PCI NCR 53c810 SCSI controller, and a 3C509 Ethernet card. All this hardware is supported by the standard Linux kernel and has been supported for some time; any distribution which does not support that test hardware should probably be considered outdated at this time. All of the tested distributions supported our test platform.
