What is a Binary File Format?

A binary file format is essentially a way of organizing or arranging an executable program, or part of a program, in a file. This includes executables (such as /bin/sh, the standard system shell), libraries (such as /lib/libc.so.5, the standard system library), and the object files from which executables and libraries are created.
