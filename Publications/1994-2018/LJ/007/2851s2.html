Getting IFS

Since this is experimental, you may not want to get it, unless all your data is backed up and it won't bother you if you have to restore it all.

If you still want to try this, it is at tsx-11.mit.edu in /pub/linux/ALPHA/ifs/. Read the file README in that directory if you want to try it out.
