Resources

ETLinux is a complete Linux distribution designed to run on small industrial computers, especially PC/104 modules. ly PC/104 modules. http://www.prosa.it/etlinux/

LEM is a small (8MB) multi-user, networked Linux version that runs on 386s. http://www.linux-embedded.com/lem.php3/

LOAF is the “Linux On A Floppy” distribution that runs on 386s. http://www.loaf.ecks.org/

uClinux is Linux for systems without MMUs. It currently supports Motorola 68K, MCF5206 and MCF5207 ColdFire microprocessors. http://www.uclinux.org/

uLinux is a tiny Linux distribution that runs on 386s. http://sunsite.auc.dk/mulinux/

ThinLinux is a minimized Linux distribution for dedicated camera servers, X-10 controllers, MP3 players and other such embedded applications. http://www.thinlinux.org/ RTLinux is a hard real-time Linux API originally developed at New Mexico Institute of Technology. http://www.rtlinux.com/ and http://www.fsmlabs.com/

RTAI is a spinoff of the RTLinux real-time API that was developed by programmers at the Department of Aerospace Engineering, Polytechnic Politecnico di Milano (DIAPM). http://www.rtai.org/

EL/IX is a proposed POSIX-based hard real-time Linux API being promoted by Red Hat. http://sourceware.cygnus.com/elix/

Linux-SRT is a “soft real-time” alternative to real-time APIs that provides performance-enhancing capabilities to any Linux program without requiring the program to be modified or recompiled. http://www.cl.cam.ac.uk/~dmi100 0/linux-srt/ FirePlug Linux is optimized as a minimal-resource firewall and routing package that works well on 486 CPUs. http://www.fireplug.net/

KOSIX is an open-source software project to develop an industry-standard public kiosk terminal operating system that is stable, secure and powerful. http://www.touchdynamics.com/kosix.html/

Linux Router Project is a “networking-centric micro-distribution” of Linux that makes it easy to build/maintain routers, access servers, thin servers, thin clients, network appliances and embedded systems (can fit on a single floppy). http://www.linuxrouter.org/

PizzaBox Linux is a Linux and Samba-based file, print and CD server designed to run from 6MB of Flash on a 486 (or higher) CPU. http://www.jrcs.co.uk/
