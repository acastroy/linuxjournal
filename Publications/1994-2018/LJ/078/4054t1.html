Script Name & Description

Check_accounts -- Checks user account for invalid shells, .rhost files, cron jobs, .hushlogin files, directory permission problems, and comments in passwd file.

Check_aliases -- Checks sendmail aliases file for misconfigurations, including non-root-owned programs aliases, include files and files with extra write permissions.

Check_anonftp -- Checkanonymous ftp for misconfigurations, including files owned by ftp and the presence of an .rhost file.

Check_cron -- Checks cron jobs for path names, references to files or executables not owned by the owner of the cron job or system accounts, and group and world-writable components.

Check_embedded -- Checks for components of paths in executables to verify that ownership and permissions do not create vulnerabilities.

Check_exports -- Checks file system exports and flags level of vulnerability due to NFS.

Check_group -- Checks Groups Ids in the passwd and group files for misconfiguration and duplications.

Check_inetd -- Checks inetd and services for misconfiguration, executable ownership problems and services vulnerablities.

Check_known -- Checks for disk storage places commonly used by intruders.

Check_netrc -- Checks .netrc and .netrcs files for ftp logins and password, and file permissions.

Check_nisplus -- Checks NIS+ password file for misconfiguration.

Check_passwd -- Checks passwd file for misconfiguration, conflicts and duplicates.

Check_path -- Checks ownership, permissions and the presence of files in PATH.

Check_perm -- Checks system files for ownership and permission vulnerabilities.

Check_printcap -- Checks ownership and permissions of printer control executables.

Check_rhosts -- Checks for .rhost files for misconfigurations and policy violations.

Check_sendmail -- Checks sendmail for misconfigurations.

Check_signature -- Checks crytographic hash signatures of system binaries against a database.

Check_system -- Configurable system check file.

Crack_run -- Runs Crack utility on passwd file.

Find_files -- Checks for unusual file and device ownership.
