The current version of RCS is 5.6.0.1. The official repository for RCS releases and beta versions is ftp.cs.purdue.edu in the /pub/RCS directory. It is also available in the file rcs-5.6.0.1.tar.gz from ftp.gnu.ai.mit.edu in /pub/gnu and should be available from any GNU mirror site, as well. The current version is maintained by Paul Eggert. Ports to other PC operating systems are available. For example, the DOS and OS/2 ports of RCS and CVS, done by Kai Uwe Rommel, are ftp-able from ftp.informatik.tu-muenchen.de in the /pub/comp/os/os2/gnu/devtools directory. Ports are also available for NT and VMS. This can be of practical value if you have Linux boxes on a network with other kinds of boxes.
