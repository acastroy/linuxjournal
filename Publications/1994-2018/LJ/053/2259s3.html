Export Elements

ExportClientHistogram: Send histogram data corresponding to the source image back to the client.

ExportClientLUT: Send LUT data back to the client.

ExportClientPhoto: Send image data back to the client. The client must choose the encoding.

ExportClientROI: Send ROI data back to the client.

ExportDrawable: Route color-map index data to a window or pixmap. This is how color and gray-scale image data is displayed in a window.

ExportDrawablePlane: Send a two-tone (one bit) image to a specific plane of a window or pixmap.

ExportLUT: Store LUT data in a server LUT resource for future use.

ExportPhotomap: Store image data in a server Photomap resource. (We use this in our example.)

ExportROI: Store ROI data in a server ROI resource for future use.
