Resources

Zope, the open-source web application server, is written and published by Zope Corporation, www.zope.com. The community and development server, where you can download the latest version, read documentation and exchange products with other users, is at www.zope.org.

The simple products in this issue would never have been built were it not for the “Boring” product (available on www.zope.org) and the excellent two-part “minimal product” tutorial written by Zope.org member MaxM. You can read the tutorial at www.zope.org/Members/maxm/HowTo/minimal_01/source.

To learn more about Python, check out the Python web site at www.python.org. Two good Python books are Learning Python (written by Mark Lutz and David Ascher, and published by O'Reilly and Associates) and Python Essential Reference (written by David M. Beazley and Guido Van Rossum, and published by New Riders).
